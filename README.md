 Netflix GPT Clone

An Overview!
A full-stack Netflix-style streaming platform built using Next.js. The goal of this project is to improve content discovery and provide a personalized viewing experience.

Features
- User authentication (Google & GitHub login)
- Responsive Netflix-like UI
- Movie browsing with categorized sections
- GPT-based recommendation system (AI integration)

Tech Stack
- Frontend: React, Next.js, TypeScript, Tailwind CSS
- Backend: Next.js API routes
- Authentication: NextAuth
- Database: MongoDB
- AI: Gemini API

⚠️ Current Status
* Core platform and UI are working
* AI recommendation feature is under debugging
* OAuth authentication is partially working and being fixed

How to Run!
1. Clone the repo
2. Run `npm install`
3. Add `.env` file
4. Run `npm run dev`
5. Open `localhost:3003`
